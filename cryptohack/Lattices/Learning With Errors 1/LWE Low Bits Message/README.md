## Description
Now we're going to jump in and decrypt a message that's hidden in the low bits of a toy LWE system. The noise is stored in the high bits.Parameters: vector space dimension nnn ciphertext modulus qqq plaintext modulus ppp (can only encrypt messages m<pm < pm<p) must have qqq, ppp be coprime Key-gen: The secret key SSS is a random element of the vector space Zqn\mathbb{Z}_q^nZqn​.
