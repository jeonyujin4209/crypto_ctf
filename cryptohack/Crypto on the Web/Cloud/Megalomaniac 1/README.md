## Description
In June 2022, a team of researchers from the Applied Crypto Group at ETH Zurich discovered a few flaws in the E2E cryptographic architecture of MEGA. This series of challenges will put you in the shoes of a Mega cloud storage server, interacting with clients logging into their personal storage. Please note that while the implementation is really close to the real one, it is not exactly the same, as the point is to illustrate the different concepts used to compromise users' private data while keeping the codebase relatively small.First, recover the user's ShareKey!Connect at socket.cryptohack.org 13408Challenge files:  - 13408.pyResources:  - Malleable Encryption Goes AwryChallenge contributed by $in
