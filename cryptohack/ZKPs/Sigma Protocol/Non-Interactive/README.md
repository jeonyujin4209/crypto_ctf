# Non-Interactive (35pts)
## Sigma Protocol

We have now shown that the Schnorr protocol for proving knowledge of a discrete logarithm relation is a Σ\SigmaΣ-Protocol! We did this by showing that it has the correct 3 move pattern, and satisfies the properties of Completeness, Special Soundness and SHVZK (Special Honest-Verifier Zero-Knowledge.
